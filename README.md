# Urdu Quran Tarjuma


<h2> Features </h2>
<ul>
  <li>Read by Surah</li>
  <li>Read by Juz</li>
  <li>Resume or Last Read</li>
  <li>Bookmark</li>
  <li>Copy</li>
  <li>Share</li>
  <li>Dark Mode</li>
  <li>Dynamic font size</li>
  <li>Jump to specific Ayat</li>
  <li>Searching</li>
</ul>
